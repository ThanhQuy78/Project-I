# Project-I: Ứng dụng quản lý khách sạn
## Tính năng chính 
1.  **Đặt phòng (Booking):**
    * Tìm kiếm phòng trống theo thời gian thực.
    * Gợi ý phòng dựa trên số lượng khách.
    * Hỗ trợ đặt nhiều phòng cùng lúc (Batch Booking).
2.  **Check-in & Check-out:**
    * Quy trình nhận phòng, xác thực CCCD.
    * Trả phòng, tự động tính toán hóa đơn và cập nhật trạng thái phòng.
3.  **Dịch vụ phòng (Room Service):**
    * Gọi món/dịch vụ (Minibar, Giặt ủi...) và ghi nợ vào phòng.
4.  **Quản lý trạng thái:**
    * Cập nhật trạng thái thủ công (Dọn phòng xong, Bảo trì...).
